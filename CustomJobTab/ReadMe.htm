Adding a custom job tab

This sample shows how to extend the ArcGIS Workflow Manager client application with your own tab.  In the sample, the new job tab shows ESRI.com.  Job tabs can be created to open portals into other applications or show job information in a specific way.

How to use

  1. Open the solution using Visual Studio.

  2. Verify the references are still intact. It may be necessary to configure the JTXUI reference again.

  3. Compile the sample project.

 

Showing the new tab in the ArcGIS Workflow Manager Application

  1. Copy the compiled .dll from bin\debug\MyCustomJobTabs.dll to Workflow Manager Install Location\Bin.

  2. Edit Workflow Manager Install Location\config\jtxuiconfig.xml in a text editor or editor of your choice.

  3. Scroll till you find the <VIEWS> section and locate the start of the <JOB PANELS>.

  4. Add a new entry for your new job tab panel.  Then end result will look something like the following:
